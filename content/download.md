## GitHub repository

The [Github Repository](https://github.com/katalyticIT/kTailor/tree/main) containes the source code,
the makefile and examples.


## Docker image

Find an overview over the available images at the [docker hub](https://hub.docker.com/repository/docker/katalytic/ktailor/general).

You may pull the latest image with the following command:

```
docker pull katalyticit/ktailor:latest
```
