# kTailor website content

[![Deploy Hugo site to Pages](https://github.com/imfing/hextra-starter-template/actions/workflows/pages.yaml/badge.svg)](https://github.com/imfing/hextra-starter-template/actions/workflows/pages.yaml)

## About

This repository contains the source for the kTailor web page at
[ktailor.dev](https://ktailor.dev). It uses [hugo](https://gohugo.io) to render
static content from markdown documents and is based on the
[Hextra](https://github.com/imfing/hextra) template.
